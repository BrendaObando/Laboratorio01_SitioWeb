# Portafolio Web Profesional

Este proyecto consiste en el desarrollo de un **sitio web de portafolio profesional** creado para presentar proyectos académicos y tecnológicos realizados durante la formación en **Ingeniería en Sistemas de Información**.

El objetivo del sitio es mostrar de manera clara y visual las habilidades, tecnologías utilizadas y experiencias en diferentes proyectos relacionados con desarrollo de software, sistemas web, hardware y prototipado.

El portafolio permite visualizar información personal, formación académica y una sección de proyectos donde se describen distintas soluciones tecnológicas desarrolladas.

---

# Objetivo del Proyecto

El propósito principal de este sitio web es:

- Presentar proyectos académicos y personales desarrollados durante la carrera.
- Mostrar las tecnologías y herramientas utilizadas en cada proyecto.
- Crear una identidad profesional en línea.
- Facilitar la presentación del trabajo realizado a profesores, empresas o futuros empleadores.

---

# Proyectos Incluidos

El portafolio presenta tres proyectos principales:

## 1 - Sistema de Gestión Municipal – Municipalidad de Nicoya

Plataforma web desarrollada para apoyar la **digitalización de procesos administrativos municipales**, permitiendo gestionar:

- Contribuyentes
- Trámites
- Permisos
- Pagos municipales

Este sistema busca mejorar la organización de la información, reducir procesos manuales y optimizar la gestión administrativa.

**Tecnologías utilizadas**

- React
- C#
- .NET
- SQL Server

**Rol en el proyecto**

Desarrollo **Full Stack**, participando en:

- Diseño de interfaz web
- Desarrollo de backend
- Modelado y creación de la base de datos

---

## 2 - Maqueta de Planta de Tratamiento de Aguas Residuales

Proyecto desarrollado como parte del curso **Arquitectura de Computadoras**.

Consistió en la construcción de una **maqueta funcional que simula una planta de tratamiento de aguas residuales**, utilizando electrónica y sistemas embebidos.

El prototipo utiliza sensores para detectar niveles de agua y controla el flujo mediante componentes electrónicos.

**Tecnologías utilizadas**

- Arduino
- Sensor ultrasónico
- Motor de bombeo de agua
- Relay
- ProtoBoard

**Rol en el proyecto**

Participación en:

- Construcción de la maqueta
- Integración de componentes electrónicos
- Pruebas del sistema

---

## 3 - Sistema Web RedSanPablo – ASADA San Pablo

Plataforma web desarrollada para apoyar la **gestión administrativa y operativa de la ASADA San Pablo de Nandayure**.

El sistema incluye módulos para:

- Gestión de usuarios
- Notificaciones
- Reportes
- Proyectos
- Control de materiales utilizados
- Solicitudes de abonados
- Reporte de averías

Además, el sistema cuenta con una **página informativa pública** donde los usuarios pueden conocer los servicios de la ASADA y realizar comentarios de forma anónima.

**Tecnologías utilizadas**

- React
- Tailwind CSS
- NestJS
- MySQL
- GitHub

**Rol en el proyecto**

Participación en el **desarrollo Frontend**, incluyendo:

- Diseño de interfaces
- Integración de módulos del sistema
- Implementación de componentes visuales
