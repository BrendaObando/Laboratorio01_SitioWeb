# CV + Portafolio (React + TS + Tailwind + TanStack Router)

Este proyecto es una adaptación del template original (Next.js App Router) a:
- React + TypeScript (Vite)
- TailwindCSS
- TanStack Router

## Correr el proyecto
```bash
npm install
npm run dev
```

## Rutas
- `/` -> Landing del CV (secciones)
- `/portafolio` -> Tarjetas de proyectos (multimedia + detalles)

## Tema (claro/oscuro)
El botón en el header guarda la preferencia en `localStorage`.
